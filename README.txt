## 📌 Bheema Institute of Technology & Science – College Website

This project is a modern, responsive college website built for **Bheema Institute of Technology & Science (BITS), Adoni**. It showcases the institution’s academic offerings, campus facilities, placements, and admission details in a clean and user-friendly interface.

The website is designed to provide students, parents, and visitors with a seamless digital experience while highlighting the college as an **AI-enabled, industry-focused institution**.

---

## 🚀 Features

* 🎯 **AI-Enabled Campus Showcase**
* 📚 **Programs Section** (B.Tech, M.Tech, Diploma)
* 📊 **Placement Statistics & Recruiters**
* 🏫 **Campus Facilities Overview**
* 🎓 **Admissions Information (EAPCET / Management / ECET)**
* 📱 **Fully Responsive Design**
* ⚡ **Modern UI with Clean Layout**
* 🔗 **Quick Navigation & Call-to-Action (Apply Now)**
